# Bank Account Management System (Java)

Console-based banking system built using Core Java. Demonstrates strong understanding of OOP concepts like inheritance, abstraction, encapsulation, and polymorphism.

## Features
- Create Savings or Current accounts
- Deposit and withdraw money
- Display account details
- View transaction history

## Concepts Used
- Java OOP (Abstract classes, Inheritance, Interfaces)
- Java Collections (`Map`, `List`)
- Basic Exception Handling
- CLI-based user input (Scanner)

## How to Run
Compile and run `Main.java` from your Java IDE or terminal.
